# Landing Page - Psicólogo y Psicoanalista

Landing page moderna para consultorio de psicología en Tres de Febrero, Buenos Aires.

## Tecnologías

- Next.js 14 (App Router)
- Tailwind CSS
- TypeScript
- Lucide React (íconos)

## Configuración

1. **Número de WhatsApp**: Edita `app/page.tsx` y reemplaza `WHATSAPP_NUMBER` con tu número real (formato: 54911XXXXXXXX, sin espacios ni guiones).

2. **Nombre**: Reemplaza "Dr. [Nombre]" en el Header y Footer por el nombre real.

3. **Matrícula**: Actualiza "MN 12345" por la matrícula real.

## Desarrollo

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Producción

```bash
npm run build
npm start
```
